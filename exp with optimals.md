Example:
(a > 10) | (b < 9) -->
[
	{
	    "a": "11",
	    "b": "9"
	},
	{
	    "a": "0",
	    "b": "0"
	},
	{
	    "a": "0",
	    "b": "9"
	}
]

Used Expressions: 
(x < 10) -->
[
	{
	    "x": "10"
	},
	{
	    "x": "0"
	}
]

(x = 0) & (y = 0) -->
[
	{
	    "x": "10",
	    "y": "11"
	},
	{
	    "x": "11",
	    "y": "10"
	},
	{
	    "x": "10",
	    "y": "10"
	}
]
    
(w >= x) & (x = y) & (y >= z) --> 
[
	{
	    "w": "0",
	    "x": "0",
	    "y": "1",
	    "z": "1"
	},
	{
	    "w": "0",
	    "x": "1",
	    "y": "1",
	    "z": "1"
	},
	{
	    "w": "0",
	    "x": "0",
	    "y": "0",
	    "z": "0"
	},
	{
	    "w": "0",
	    "x": "0",
	    "y": "0",
	    "z": "1"
	}
]

	{

	}
]
