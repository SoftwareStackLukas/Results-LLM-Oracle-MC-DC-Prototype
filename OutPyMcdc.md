	{
	    "x": "11",
	    "y": "9"
	},
	{
	    "x": "0",
	    "y": "0"

(x < 10) -->
```javascript
[
	{
	    "x": "10"
	},
	{
	    "x": "0"
	}
]
```

(x = 10) & (y = 10) -->
```javascript
[
	{
	    "x": "10",

	},
	{
	    "x": "11",
	    "y": "10"
	},
	{
	    "x": "10",
	    "y": "10"
	}
]
```
